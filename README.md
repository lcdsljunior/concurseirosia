# 🤖 Mentor de Concursos - IA com Gemini Pro

Este repositório faz parte do meu estudo de caso sobre **Desenvolvimento de Agentes de IA** aplicados à educação para concursos públicos.

## 📖 O Estudo de Caso
O desafio foi criar uma ferramenta que não apenas responda perguntas, mas que siga uma metodologia de estudo ativa (Explicação + Base Legal + Exercício).

## 🛠️ Tecnologias Utilizadas
- **Python**: Linguagem base.
- **Google Gemini API**: Modelo de linguagem (LLM).
- **Python-Dotenv**: Gestão de variáveis de ambiente e segurança.
- **GitHub**: Versionamento e documentação.

## 🚀 Como Executar
1. Instale as dependências: `pip install -r requirements.txt`
2. Configure seu `.env` com a API Key do Google.
3. Execute `python app.py`.
